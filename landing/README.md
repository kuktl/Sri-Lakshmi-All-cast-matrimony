# Telugu Matchmakers

India's Largest Network of Trusted Matchmakers - A landing page website for Telugu Matchmakers platform.

## Features

- **Hero Section** - Eye-catching hero with CTAs
- **Why Telugu Matchmakers** - Feature highlights with icons
- **How It Works** - 4-step process visualization
- **About Us** - Company information
- **Become a Partner** - Partner registration section
- **Success Stories** - Testimonials from couples
- **Contact Us** - Contact form with phone mockup
- **Footer** - Links and social media

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Poppins, Great Vibes)
- Font Awesome Icons

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser, or serve with any static server:

```bash
python3 -m http.server 8080
```

3. Visit `http://localhost:8080`

## Design

- Color scheme: Maroon, Gold, Cream
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll animations
- Mobile hamburger navigation
